## 6. 已知问题与解决方案

### 6.1 已知问题

| 问题 | 影响范围 | 严重程度 | 状态 |
|-----|---------|---------|------|
| Safari 移动端拖拽体验略差 | iOS 设备 | 低 | 已优化触摸反馈 |
| 极小屏幕下座位网格缩放过度 | < 320px | 低 | 已设置最小缩放 50% |
| 部分旧版浏览器不支持 CSS 变量 | IE11 等 | 不影响 | 不支持旧版浏览器 |

### 6.2 优化建议

1. **进一步优化触摸拖拽**：考虑使用第三方库（如 Sortable.js）增强移动端拖拽体验
2. **添加离线支持**：使用 Service Worker 实现离线缓存
3. **图片懒加载**：如果未来添加图片资源，建议实现懒加载
